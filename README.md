# Kanban Task Board

## Overview

A modern Kanban Task Management application built using React.js and Vite.

This application helps users organize tasks across three stages:

* To Do
* In Progress
* Done

## Live Demo

https://kanban-task-board-black.vercel.app/

## Features

* Add Tasks
* Edit Tasks
* Delete Tasks
* Move Tasks Between Columns
* Drag and Drop Support
* Priority Levels (High, Medium, Low)
* Search Functionality
* Local Storage Persistence
* Responsive Design

## Tech Stack

* React.js
* Vite
* CSS3
* dnd-kit

## Installation

npm install

npm run dev
